# Обзор Backend

Backend часть приложения реализована на фреймворке FastAPI (Python) и отвечает за обработку запросов от клиентской части, управление данными, их хранение и взаимодействие с внешними инструментами для генерации документов.

## Основные технологии и библиотеки:

*   **FastAPI**: Современный, быстрый (высокопроизводительный) веб-фреймворк для создания API на Python 3.7+.
*   **SQLAlchemy**: Мощная библиотека SQL Toolkit и Object Relational Mapper (ORM), используемая для взаимодействия с базой данных.
*   **Alembic**: Инструмент для миграции баз данных, интегрированный с SQLAlchemy, позволяющий управлять изменениями схемы БД.
*   **Pydantic**: Библиотека для валидации данных и настроек с помощью Python type hints. Используется для определения схем запросов и ответов API, а также для валидации данных моделей.
*   **Uvicorn**: ASGI-сервер, используемый для запуска FastAPI приложения.
*   **Pandoc**: Утилита для конвертации документов из одного формата в другой. Используется для преобразования Markdown в Typst.
*   **Typst CLI**: Система разметки документов (как LaTeX, но современнее и проще), используется для компиляции файлов `.typ` в высококачественные PDF-документы.

## Структура директорий Backend

Директория `app/` организована по принципу разделения ответственности:

*   `app/api/`: Определения API-эндпоинтов, обработчики запросов.
    *   `app/api/generate.py`: Основная логика генерации документов (PDF, `.typ`).
    *   `app/api/profiles.py`: Управление профилями стилей.
    *   `app/api/elements/`: Эндпоинты для управления стилями различных элементов.
*   `app/models/`: Модели данных.
    *   `app/models/pydantic/`: Pydantic-модели для валидации данных.
    *   `app/models/sqlalchemy/`: SQLAlchemy ORM-модели, описывающие структуру базы данных.
*   `app/repositories/`: Репозитории для абстрагирования доступа к данным.
*   `app/utils/`: Вспомогательные утилиты.
    *   `app/utils/typst_preamble.py`: Логика формирования Typst-преамбулы на основе стилей профиля.
    *   `app/utils/pandoc_converter.py`: Интеграция с Pandoc для конвертации Markdown.
    *   `app/utils/typst_compiler.py`: Интеграция с Typst CLI для компиляции Typst в PDF.
*   `app/database.py`: Настройка подключения к базе данных и управление сессиями.
*   `app/deps.py`: Вспомогательные функции для внедрения зависимостей FastAPI.
*   `app/main.py`: Главный файл приложения, инициализация FastAPI.
