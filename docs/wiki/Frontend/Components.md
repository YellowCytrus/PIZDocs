# Основные Frontend Компоненты

Frontend приложение состоит из ряда Vue-компонентов, каждый из которых выполняет определенную роль в пользовательском интерфейсе. Здесь описаны ключевые компоненты, их назначение и взаимодействие.

## Макет редактора

### [EditorLayout.vue](<../../frontend/src/components/EditorLayout.vue>)

Главный компонент макета, который организует структуру страницы редактирования. Он использует `splitpanes` для создания изменяемых по размеру областей для различных частей интерфейса.

*   **`LeftToolbar`**: Боковая панель с инструментами для вставки элементов в Markdown.
*   **`MdEditor`**: Основной компонент Markdown-редактора.
*   **`TypstPreview`**: Компонент для отображения живого предпросмотра Typst-документа.
*   **`RightPanel`**: Боковая панель с настройками профилей и стилей (подробнее описан в [Редакторе стилей](StyleEditor.md)).

### [LeftToolbar.vue](<../../frontend/src/components/LeftToolbar.vue>)

Боковая панель инструментов слева от редактора. Предоставляет кнопки для быстрой вставки Markdown-синтаксиса (например, заголовков, математических формул) в `MdEditor`.

### [MdEditor.vue](<../../frontend/src/components/MdEditor.vue>)

Компонент Markdown-редактора, основанный на библиотеке `md-editor-v3`. Он позволяет пользователю вводить Markdown-текст и поддерживает загрузку изображений.

*   **`editorStore`**: Интегрируется с `useEditorStore` для синхронизации содержимого редактора и имени файла.
*   **Загрузка изображений (`onUploadImg`)**: Обрабатывает загрузку изображений через Backend API (`POST /upload-image`), преобразуя загруженные файлы в URL-адреса, которые можно вставить в Markdown. 
*   **Функциональность обрезки изображений**: Включает модальное окно для обрезки изображений перед загрузкой.
*   **Drag & Drop**: Поддерживает перетаскивание Markdown или текстовых файлов для загрузки в редактор.
*   **MathLive Formula Builder**: Кастомный dropdown-тулбар (`MdEditorMathToolbar.vue`) открывает модалку конструктора формул (`MathLiveBuilderModal.vue`) с копированием/вставкой LaTeX.

### [MathLiveBuilderModal.vue](<../../frontend/src/components/MathLiveBuilderModal.vue>)

Модальное окно конструктора формул на базе `math-field`.

*   **Редактирование**: Пользователь вводит формулу в raw-виде (без немедленной трансформации во время ввода).
*   **Экспорт**: Перед copy/insert применяется `sanitizeLatex()` для совместимости с Pandoc → Typst.
*   **Виртуальная клавиатура**: Управляется вручную (`virtual-keyboard-mode="manual"`) с управлением фокусом и стабильным открытием.

Подробнее: [MathLive Formula Builder](MathLiveBuilder.md)

### [TypstPreview.vue](<../../frontend/src/components/TypstPreview.vue>)

Компонент для отображения живого предпросмотра Typst-документа. Он получает полный Typst-исходник (сформированный из Markdown и стилей) и рендерит его на HTML Canvas с помощью WebAssembly-модуля `typst.ts`. Подробнее описан в [Пайплайне рендеринга](RenderingPipeline.md).

### [RightPanel.vue](<../../frontend/src/components/RightPanel.vue>)

Правая панель, содержащая основные настройки профилей и некоторые глобальные стили. Подробнее описан в [Редакторе стилей](StyleEditor.md).

## Компоненты редактора титульных страниц

Эти компоненты используются на странице `PageEditor.vue` для создания и редактирования титульных страниц.

### [page-editor/Canvas.vue](<../../frontend/src/components/page-editor/Canvas.vue>)

Основной холст для отображения и взаимодействия с элементами титульной страницы. Рендерит текстовые блоки, переменные и линии, а также предоставляет функциональность перетаскивания, изменения размера и привязки элементов.

*   **`usePageEditorStore`**: Взаимодействует с Pinia-стором для получения и обновления элементов титульной страницы.
*   **`useCanvas`**: Компосабл для управления масштабированием и панорамированием холста.
*   **Механизмы взаимодействия:** Обрабатывает события мыши (`mousedown`, `mousemove`, `mouseup`, `wheel`) для реализации перетаскивания элементов, изменения их размеров и прокрутки/масштабирования.
*   **Привязка (Snapping):** Использует утилиты из `frontend/src/utils/snap.ts` для привязки элементов к сетке или другим элементам на холсте.
*   **Рендеринг:** Использует Canvas 2D API для отрисовки всех элементов (текстовые поля, переменные, линии, сетка, направляющие) на холсте.

### [page-editor/Properties.vue](<../../frontend/src/components/page-editor/Properties.vue>)

Панель свойств, отображающая настройки для выбранного элемента на холсте `Canvas.vue`. Позволяет редактировать содержимое текста, имена и значения переменных, а также стили текста.

*   **`usePageEditorStore`**: Получает информацию о выбранном элементе и обновляет его свойства.
*   **`StyleFieldInput`**: Используется для редактирования текстовых стилей (шрифт, размер, цвет и т.д.) для текстовых элементов и переменных.

### [page-editor/Sidebar.vue](<../../frontend/src/components/page-editor/Sidebar.vue>)

Боковая панель редактора титульных страниц, содержащая инструменты для добавления новых элементов на холст (текстовые поля, переменные, линии), а также настройки страницы (формат бумаги, отступы) и параметры сетки/привязки.

*   **`usePageEditorStore`**: Используется для добавления новых элементов и изменения настроек страницы/холста.
*   **`PAPER_PRESETS`**: Предварительно определенные форматы бумаги для быстрого выбора.
