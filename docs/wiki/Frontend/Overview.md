# Обзор Frontend

Frontend часть приложения представляет собой одностраничное веб-приложение (SPA), разработанное на Vue.js 3 с использованием TypeScript, Vite для сборки, Pinia для управления состоянием и Tailwind CSS для стилизации. Она обеспечивает взаимодействие пользователя с системой, предоставляя редактор Markdown, инструменты для управления стилями и предпросмотр документов.

## Основные технологии и библиотеки:

*   **Vue.js 3**: Прогрессивный JavaScript-фреймворк для построения пользовательских интерфейсов. Используется Composition API для организации логики.
*   **TypeScript**: Язык программирования, добавляющий статическую типизацию к JavaScript, что повышает надежность и упрощает разработку.
*   **Vite**: Быстрый сборщик фронтенда, обеспечивающий мгновенный холодный старт сервера разработки и быструю горячую замену модулей (HMR).
*   **Pinia**: Легковесное и интуитивно понятное решение для управления состоянием Vue.js приложений, является официальной рекомендацией для Vue 3.
*   **Tailwind CSS**: Утилитарный CSS-фреймворк, позволяющий быстро строить пользовательские интерфейсы, непосредственно указывая классы в разметке.
*   **`typst.ts`**: WebAssembly-модуль, предоставляющий функциональность рендеринга Typst-документов непосредственно в браузере. Используется для реализации живого предпросмотра.
*   **`@vueuse/core`**: Коллекция Composition API утилит для Vue.js, упрощающая работу с различными аспектами (например, `useFileDialog`, `useLocalStorage`).

## Структура директорий Frontend

Директория `frontend/src/` организована для ясности и разделения ответственности:

*   `frontend/src/views/`: Компоненты, представляющие собой целые страницы или основные разделы приложения (например, `Editor.vue`, `Styles.vue`).
*   `frontend/src/components/`: Переиспользуемые UI-компоненты (например, `MdEditor.vue`, `TypstPreview.vue`, компоненты для редактора стилей).
*   `frontend/src/composables/`: Vue Composition API функции для инкапсуляции и переиспользования логики (например, `usePandoc`, `useStyles`).
*   `frontend/src/stores/`: Pinia-сторы для централизованного управления состоянием приложения (например, `editor.ts`, `profiles.ts`, `preview.ts`).
*   `frontend/src/types/`: TypeScript-файлы с определениями типов данных, включая контракты с API (`api.ts`).
*   `frontend/src/utils/`: Вспомогательные утилиты, не связанные с Vue-специфичной логикой (например, `unitConversion.ts`).
*   `frontend/src/router/index.ts`: Настройка Vue Router для клиентской маршрутизации.
*   `frontend/src/config.ts`: Конфигурация Frontend-приложения (например, базовый URL API).
*   `frontend/src/main.ts`: Точка входа в приложение Vue, инициализация плагинов и корневого компонента.

## Основные функциональные блоки

1.  **Редактор документов:** Предоставляет интерфейс для написания Markdown, выбора профилей стилей и титульных страниц.
2.  **Редактор стилей:** Позволяет пользователям просматривать и изменять параметры стилей для различных элементов документа, сохраняя их в профили.
3.  **Живой предпросмотр:** Отображает рендеринг Typst-документа в реальном времени, используя `typst.ts` для компиляции и отображения на Canvas.
4.  **Взаимодействие с API:** Frontend отправляет запросы к Backend API для сохранения/загрузки данных и генерации PDF/Typst файлов.
