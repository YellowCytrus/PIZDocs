# Конвертация единиц измерения на Frontend

Frontend приложение предоставляет удобный интерфейс для работы с различными единицами измерения при настройке стилей. Для обеспечения точности и гибкости реализован механизм конвертации, позволяющий пользователям вводить значения в привычных им единицах, которые затем внутренне преобразуются в миллиметры (mm) для единообразной обработки.

## Утилита `unitConversion.ts` ([`frontend/src/utils/unitConversion.ts`](../../frontend/src/utils/unitConversion.ts))

Этот модуль содержит основные функции для преобразования значений между различными единицами измерения.

### Поддерживаемые единицы и коэффициенты

*   `mm` (миллиметры): Базовая единица, коэффициент 1.
*   `cm` (сантиметры): `10 mm`.
*   `in` (дюймы): `25.4 mm` (международный дюйм).
*   `pt` (пункты): `1/72 in = 25.4/72 mm`.
*   `em`: Относительная единица, зависящая от текущего размера шрифта документа. Требует `fontSizeMm` в `Context`.

### `toMm(value: number, unit: Unit, ctx?: Context): number`

Преобразует заданное значение из указанной единицы (`unit`) в миллиметры.

*   Для `em` требуется контекст (`ctx`), содержащий `fontSizeMm` (размер шрифта документа в мм).

### `fromMm(valueMm: number, unit: Unit, ctx?: Context): number`

Преобразует заданное значение из миллиметров (`valueMm`) в указанную единицу (`unit`).

*   Также требует `Context` для конвертации из/в `em`.

### `convert(value: number, from: Unit, to: Unit, ctx?: Context): number`

Универсальная функция для конвертации значения из одной единицы в другую, используя `toMm` и `fromMm` как промежуточные шаги.

### `ptToMm(valuePt: number): number`

Специализированная функция для конвертации пунктов в миллиметры, используемая для получения контекста `fontSizeMm`.

## Компонент `UnitInput.vue` ([`frontend/src/components/styles/UnitInput.vue`](../../frontend/src/components/styles/UnitInput.vue))

Этот компонент предоставляет пользовательский интерфейс для ввода числовых значений с возможностью выбора единицы измерения. Он интегрируется с `unitConversion.ts` для выполнения всех преобразований.

**Ключевые особенности:**

*   **Слайдер и числовой ввод:** Компонент включает в себя слайдер и текстовое поле для ввода числового значения. При изменении значения через любой из этих элементов, оно сразу же конвертируется в миллиметры и передается родительскому компоненту через события `preview` (для отображения промежуточных значений) и `commit` (для фиксации значения).
*   **Выбор единицы:** Пользователь может выбрать желаемую единицу измерения из выпадающего списка (`select`). При выборе новой единицы, отображаемое значение пересчитывается.
*   **Контекст для `em`:** Если выбранная единица — `em`, компонент проверяет наличие `context.fontSizeMm`. Если контекст отсутствует или некорректен, отображается сообщение об ошибке, и ввод блокируется.
*   **Динамические min/max/step:** Минимальные, максимальные значения и шаг ввода также конвертируются в выбранную единицу отображения, чтобы обеспечить корректное поведение слайдера и числового поля.

## Интеграция в редактор стилей

`UnitInput.vue` используется внутри [`StyleFieldInput.vue`](../../frontend/src/components/styles/StyleFieldInput.vue) для полей, которые поддерживают конвертацию единиц измерения. `StyleFieldInput` определяет, является ли поле конвертируемым с помощью `isMmConvertibleField` ([`frontend/src/config/styleFields.ts`](../../frontend/src/config/styleFields.ts)), и передает ему необходимый контекст.
